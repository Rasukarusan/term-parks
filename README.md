Terminal Parks
====

![demo.gif](https://user-images.githubusercontent.com/17779386/116769691-f188bd80-aa78-11eb-89db-401f801f917a.gif)

## Description

connect your terminal from browser.

## Install

```sh
cd packages/client
yarn
cd ..

cd packages/server
yarn
cd ..

yarn dev

open "http://localhost:3000/terminal"
```
