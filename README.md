Terminal Parks
====

![demo.gif](https://user-images.githubusercontent.com/17779386/116769656-a40c5080-aa78-11eb-9495-c02b4a0568ee.png)

## Description

connect your terminal from browser.

## Install

```sh
cd packages/client
yarn
cd ..

cd packages/server
yarn
cd ..

yarn dev

open "http://localhost:3000/terminal"
```
